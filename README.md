### Functions

- Add new reviews
- Edit reviews
- Delete reviews
- Add/Edit/Delete new categories
- Search Review
- Sort Review
- Random restaurant
- Add Favourite
- Backup/Restore database
- Light/Dark theme
- Display method
- Daily Reminder
- Different Languages
